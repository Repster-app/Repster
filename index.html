Workout data that stays useful after the set is done.

Repster keeps entry quick during training, then turns the history into records, trends, calendar detail, and exercise-level context.

Fast workout entry

Log sets, edit workouts, track rest, and move through exercises without slowing down the session.

Smart suggestions

Use recent performance and fatigue-aware context to guide the next working weight.

Records and history

Surface PRs, previous sets, exercise history, and calendar detail where they are most useful.

Charts and import

Review progress over time and bring existing workout history forward from CSV exports.

Built for repeated use, not one-off demos.

The core screens prioritize scanning, editing, and comparing workout information with minimal friction.

Repster is being built as a focused iOS training log.

This public repository hosts the Repster website. The iOS app source code stays in a separate private repository.
