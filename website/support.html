Frequently Asked Questions

Smart Suggestions estimate the right weight for each set based on your recent performance and current fatigue level. The system looks at your best recent e1RM, accounts for fatigue accumulated during the session, and factors in rest between sets. Over time, it learns your personal fatigue patterns and becomes more accurate. You can configure target reps, target RIR, and other settings in Settings > Prescription Settings.

After finishing a workout, you can rate each exercise's suggestions as "less aggressive," "about right," or "more aggressive." This feedback helps the adaptive fatigue learning system adjust its model for that specific exercise, making future suggestions more aligned with how your body responds.

Reppo automatically evaluates every completed set against your full history for that exercise. If a set represents a new best weight at a given rep count (or higher), it's flagged as a PR. The system uses a frontier algorithm — meaning a 5-rep PR at 100kg is only "current" if you don't also have a 6-rep PR at 100kg or higher.

Yes. Reppo supports CSV import in the FitNotes format. Go to Settings > Import Data to select your CSV file. The import process previews your data before committing, creates any new exercises automatically, and rebuilds all stats and PRs after completion.

Go to Settings > Export Backup to create a full backup file containing all your workouts, exercises, sets, templates, and fatigue learning data. Save it to your preferred location. To restore, use Settings > Restore Backup and select a previously exported file.

Reppo allows you to log up to 5 workouts for free. After that, a subscription is required for full access. The subscription unlocks unlimited workout logging and all features. Subscriptions are managed through the App Store.

No. All your data is stored locally on your device. Reppo does not have user accounts, servers, or cloud sync. The only external connection is to RevenueCat for subscription verification. See our Privacy Policy for full details.

The AI template generator creates a prompt based on your exercise library that you can copy and paste into ChatGPT. ChatGPT returns a JSON response, which you paste back into Reppo to preview and import as a template. Reppo does not connect to any AI service directly — you handle the conversation with ChatGPT yourself.

Reppo supports multiple tracking types to cover different exercise styles: weight + reps, duration only, duration + distance, weight + distance, weight + reps + duration, and custom. You set the tracking type when creating an exercise, and it determines which fields appear in the set entry UI.
