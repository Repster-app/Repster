# Creator And Press Kit

## One-Paragraph Description

Repster is an iOS workout tracker for lifters who want fast set logging and training history that stays useful during the session. It supports reusable templates, previous-set context, PRs, charts, rest timers, Live Activities, Smart Suggestions, CSV import, and local-first workout data with exportable backups.

## Short Description

Fast workout tracking for lifters: log sets, start from templates, see previous performance, track PRs, and review progress charts.
