# Source Assets

This folder contains local source assets for the Repster marketing renderer.

## Logo

- `logo/repster-logo.png`: Copied from the app logo source in the repository.

## Screenshots

- `screenshots/active-sets.png`: Local simulator screenshot of the active workout set logger.
- `screenshots/active-history.png`: Local simulator screenshot of previous performance inside an active workout.
- `screenshots/active-chart.png`: Local simulator screenshot of active workout chart context.
- `screenshots/site-home.png`: Downloaded from the published Repster site.
- `screenshots/site-templates.png`: Downloaded from the published Repster site.
- `screenshots/site-charts.png`: Downloaded from the published Repster site.

When final App Store captures are available, replace these files with same-size PNGs and rerun `marketing/tools/render_marketing_assets.swift`.
