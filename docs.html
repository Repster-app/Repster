A practical guide to the main Repster workflows: logging workouts, using templates, reviewing records, and understanding progress.

Home Dashboard

The home screen is the starting point for training. It shows weekly activity, recent workouts, recent personal records, and the fastest path into a new session.

Start workout

Begin with an empty workout, copy a previous session, or start from a saved template.

Recent activity

Recent records and workouts keep progress visible without needing to search through history.

Active Workout

The active workout screen is built for fast set entry. Exercise tabs keep the current session organized while set rows handle weight, reps, RIR, completion, and unilateral tracking where needed.

  • Sets: log the current exercise.
  • History: review previous sessions for the exercise.
  • PRs: see personal records while training.
  • Charts: inspect exercise-level trends.

Templates

Templates let you save workout structures and reuse them. Tap a template to start fast, create one manually, save one from a completed workout, or import a shared template file.

Template rows show exercise count, set count, target areas, and quick actions for management.

Personal Records

Repster evaluates completed sets against your exercise history and flags meaningful records. PRs appear during workouts, on the home dashboard, and in exercise detail views.

The app distinguishes current records, matched records, previous records, and records that have been surpassed by stronger performances.

Charts and Analytics

The charts tab shows training volume, workout trends, and exercise progress over different time ranges. Breakdown charts help you see how volume is distributed across muscle groups or exercises.

  • Breakdown: volume distribution by category.
  • Workouts: session frequency and workload over time.
  • Exercises: individual exercise performance trends.

Import, Export, and Backup

Repster supports importing workout history from CSV formats and creating full backups. Imports preview and validate data before it is committed, then rebuild stats and records afterward.

Backups can be exported and restored from the Settings screen.

Settings

Settings control units, estimated one-rep-max formula, rest timing, import/export, bodyweight entries, and prescription behavior where available.

Bodyweight entries are stored on-device and can be used for effective weight calculations on bodyweight-factor exercises.

Subscription

Repster includes a free tier for trying the app. Full access unlocks unlimited workout logging and is managed through Apple's App Store subscriptions.
